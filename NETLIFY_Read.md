# 🚀 Розгортання Розумника на Netlify + вставка на Wix

---

## Крок 1 — Реєстрація на Netlify

1. Зайдіть на https://netlify.com
2. Натисніть **"Sign up"** → оберіть **"Sign up with Email"**
3. Заповніть email і пароль → підтвердіть email

> Netlify безкоштовний. Карта не потрібна.

---

## Крок 2 — Завантаження проекту

1. Після входу ви побачите Dashboard
2. Знайдіть внизу сторінки блок:
   **"Want to deploy a new site without connecting to Git?"**
3. Під ним є поле з написом:
   **"Drag and drop your site output folder here"**
4. **Перетягніть папку `rozumnyk_v2`** прямо у це поле

> ⚠️ Перетягуйте саму ПАПКУ, не zip-архів і не окремі файли.
> У папці повинні бути: rozumnyk.html, index.html, netlify.toml, TESTS/

5. Netlify автоматично завантажить всі файли і через 10-30 секунд
   видасть URL вигляду: `https://назва-назва-123456.netlify.app`

---

## Крок 3 — Перевірка

1. Відкрийте отриманий URL у браузері
2. Переконайтесь що каталог тестів завантажується
3. Пройдіть один тест повністю

---

## Крок 4 — Своя назва (опційно)

За замовчуванням Netlify дає випадкову назву. Щоб змінити:

1. У Dashboard клікніть на ваш сайт
2. **Site configuration** → **Site details** → **Change site name**
3. Введіть: `rozumnyk` (або іншу зручну назву)
4. URL стане: `https://rozumnyk.netlify.app`

---

## Крок 5 — Вставка на Wix

1. Відкрийте редактор Wix
2. Зліва: **Add Elements** → **Embed Code** → **Embed HTML**
3. Перетягніть блок на сторінку
4. Клікніть на блок → **"Enter Code"**
5. Вставте наступний код (замініть URL на ваш):

```html
<iframe 
  src="https://rozumnyk.netlify.app/rozumnyk.html"
  width="100%" 
  height="750"
  frameborder="0"
  scrolling="auto"
  allow="fullscreen">
</iframe>
```

6. Натисніть **Apply**
7. Налаштуйте висоту блоку на Wix: мінімум **750px**
8. Збережіть та опублікуйте сайт

> 💡 Це лише 210 символів — добре вкладається в ліміт 15 000 Wix.

---

## Як додавати нові тести

### Варіант А — через Netlify Dashboard (рекомендовано)

1. Зайдіть на netlify.com → ваш сайт
2. **Deploys** → **Drag & drop** нову версію папки `rozumnyk_v2`
   (з новими xlsx і оновленим manifest.json)
3. Netlify автоматично оновить сайт. Wix нічого міняти не потрібно.

### Варіант Б — через GitHub (для регулярних оновлень)

1. Створіть безкоштовний акаунт на github.com
2. Завантажте папку як репозиторій
3. Підключіть репозиторій до Netlify (Import from Git)
4. Далі: додали xlsx → push до GitHub → Netlify оновлюється автоматично

## Часті питання

**Чому не відкривається каталог тестів?**
Перевірте що у папці є файл `TESTS/manifest.json` і він не порожній.

**Чи можна використовувати свій домен?**
Так. Netlify безкоштовно підключає власний домен:
Site configuration → Domain management → Add domain.

**Чи безпечно зберігати тести на Netlify?**
Так — всі файли публічно доступні за URL, але це статичні файли xlsx.
Приватних даних учнів там немає (результати ніде не зберігаються).
